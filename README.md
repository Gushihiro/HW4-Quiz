# HW4-Quiz

This is a quiz with questions based on Square Enix content. The quiz iterates through an array with 4 objects containing a question, four answers, and a correct answer which is used to match with the user's choice. Correct answers will push through the quiz while incorrect answers will lower the timer on the quiz. Completing the quiz allows the user to write his name down for the leader board which shows in the console.
